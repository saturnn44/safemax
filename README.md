# safemax
safemax is a MAX Messenger client with some dangerous permissions deleted.

Deleted: **camera, gallery, microphone, contact list, biometry and location access.**

Guide: install* the file on your device.

* **.apks file cannot install without a proper program. For installation use SAI** `(https://play.google.com/store/apps/details?id=com.mtv.sai)`

I do not take any right for MAX messenger from it's authors/developers.
