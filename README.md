## safemax
safemax is a MAX Messenger client with some dangerous permissions deleted.

Deleted: **camera, gallery, microphone, contact list, biometry and location access.**

Guide: install* the file on your device.

* **.apks files cannot be installed without a proper program. For installation use 'SAI'** ([Google Play](https://play.google.com/store/apps/details?id=com.mtv.sai)/[F-DROID](https://f-droid.org/ru/packages/com.aefyr.sai.fdroid/))
